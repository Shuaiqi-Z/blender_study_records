这是 Blender 动画制作中**最核心、最专业**的一个面板，叫做：

# **Graph Editor (图形编辑器 / 曲线编辑器)**

如果说刚才的“动画摄影表 (Dope Sheet)”是决定动作在**什么时候**发生，那这个面板就是决定动作**是怎么发生的**（快还是慢，顺滑还是生硬）。

------

### 1. 它到底是干嘛的？

它用**曲线图**来表示数值的变化。

- **横轴（左右）：** 代表**时间**（帧数）。
- **纵轴（上下）：** 代表**数值**（比如移动了多少米、旋转了多少度）。
- **曲线的坡度：** 代表**速度**。
  - **坡度越陡：** 动作越快。
  - **坡度越平：** 动作越慢。
  - **水平直线：** 动作静止。

### 2. 为什么需要它？（告别“机器人”动作）

你在 Dope Sheet 里打两个关键帧，物体默认是“匀速”或者简单的“慢进慢出”运动的。但如果你想做出更有质感的动画，必须来这里：

- **调整节奏 (Easing)：** 你想让易拉罐“嗖”地一下飞进来，然后慢慢停下？你需要在这里把曲线调成“先陡峭后平缓”。
- **做弹性动画 (Bounce)：** 你想让易拉罐落地时弹两下？你需要在这里把曲线画成波浪形。
- **精确控制：** 你可以直接拖动曲线上的点，把位置精确到小数点后几位。

### 3. 图里的关键元素

- **有颜色的线：** 对应不同的轴向。
  - 🔴 **红线：** X 轴数据。
  - 🟢 **绿线：** Y 轴数据。
  - 🔵 **蓝线：** Z 轴数据。
- **控制手柄 (Bezier Handles)：**
  - 选中曲线上任何一个黑点（关键帧），你会看到它旁边伸出两根“杠杆”。
  - **旋转/拉长**这个杠杆，就能改变曲线的弯曲程度，从而改变动画的“加速度”。

### 💡 总结

这是动画师的“调音台”。

**新手**在 Dope Sheet 里调时间（让动作对上拍子）；

**高手**在 Graph Editor 里调曲线（让动作有重量感和生命力）。

你在做产品动画时，如果要让产品转动得非常优雅（比如丝滑地开始转动，又丝滑地停下），就全靠在这里调节曲线的手柄了！

#  他是用曲率来代表速度？直线速度就是0?

这里有一个非常关键的概念误区，我来帮你彻底理清。

答案是：**“坡度”代表速度，“曲率”代表加速度。**

至于“直线速度是不是 0”？**只有“水平的横直线”速度才是 0。**

------

### 1. 怎么看速度？看“陡峭程度” (Slope)

请把这条曲线想象成一座**山坡**：

- **陡峭的山坡（/）：** 数值变化快 -->**速度快**。
- **平缓的山坡（—）：** 数值变化慢 --> **速度慢**。
- **完全水平的平地（—）：** 数值没有变化 --> **速度为 0（静止）**。

> [!IMPORTANT]
>
> **所以回答你的问题：**
>
> 如果是一条**斜着的直线**（/），它的速度**不是 0**，而是**匀速运动**（就像汽车开了定速巡航，不快也不慢，一直稳定地动）。

### 2. 怎么看曲率？看“油门和刹车” (Acceleration)

曲率（弯曲的程度）代表的是**速度的变化**：

- **弯曲的线：** 说明速度在变。
  - **越来越陡：** 说明在**加速**（踩油门）。
  - **越来越平：** 说明在**减速**（踩刹车）。
- **S 型曲线（Bezier）：** 这是动画里最常用的。
  - 开始是平的（静止） --> 慢慢变陡（加速） --> 中间最陡（最快） --> 慢慢变平（减速） --> 最后是平的（停下）。
  - 这就是所谓的 **“慢进慢出” (Ease In - Ease Out)**。

### 总结一张图：

| **线的样子**     | **物理含义**        | **实际感觉**                                  |
| ---------------- | ------------------- | --------------------------------------------- |
| **水平直线 (—)** | **速度 = 0**        | 车停在路边不动。                              |
| **斜直线 (/)**   | **匀速**            | 车在高速上定速 100km/h 巡航，没有推背感。     |
| **弯曲线 (S)**   | **变速 (有加速度)** | 红绿灯起步（从 0 加速）或者刹车（从快变慢）。 |

所以在 Graph Editor 里，你调整手柄让曲线弯曲，其实就是在控制**什么时候踩油门，什么时候踩刹车**。

#  这两个部分的 扳手，勾选框，锁，都是一个意思吗？

![image-20260123180324046](C:\Users\Administrator\AppData\Roaming\Typora\typora-user-images\image-20260123180324046.png)

这是一个非常好的观察！

简单回答：**只有“锁（🔒）”是一样的意思，其他的图标意思不同。**

左边的（Dope Sheet）主要是控制“动画播不播放”，而右边的（Graph Editor）主要是控制“曲线给不给你看”。

以下是详细对比（根据你截图中的图标）：

### 1. 相同的图标：锁 (Lock) 🔒

- **两边都有，功能完全一样。**
- **作用：** **“只读模式”**。
- **场景：** 当你点亮它，这个通道就被锁死了。你无法移动关键帧，也无法修改数值。这是为了防止你手滑误触调好的动作。

------

### 2. 左边（动画摄影表 Dope Sheet）独有的：

这是你截图左边红框里的图标：

- **🔧 扳手 (Wrench / Modifiers):**
  - **意思：** **“修改器状态”**。
  - **作用：** 如果你给曲线加了“噪声 (Noise)”或“循环 (Cycles)”等修改器，这个扳手会变亮。点击它可以快速开关修改器的效果。
- **✅ 勾选框 (Checkmark / Mute):**
  - **意思：** **“静音/禁用”**。
  - **作用：** **这是真关。** 如果你把 `X Location` 前面的勾去掉，这个物体在 X 轴上就**彻底不动了**（虽然关键帧还在，但不起作用）。

------

### 3. 右边（曲线编辑器 Graph Editor）独有的：

这是你截图右边红框里的图标（仔细看，图标其实不一样）：

- **👁️ 眼睛 (Eye / Visibility):**
  - **意思：** **“隐藏曲线”**。
  - **作用：** **这是假关。** 它只是把那条红/绿/蓝的曲线从屏幕上**藏起来**，不让你看见，**但动画依然会正常播放，物体依然会动**。
  - *场景：当你只想专心调 X 轴的曲线，觉得 Y 轴和 Z 轴的线太乱了，就可以把 Y 和 Z 的眼睛闭上。*
- **🖱️ 箭头 (Cursor / Selection):**
  - **意思：** **“是否可被选中”**。
  - **作用：** 如果你点掉它，你就无法用鼠标选中这条曲线了。这也是为了防止误操作。

### 总结对照表

| **图标** | **名字**   | **所在区域**            | **它的作用**            | **动画还会动吗？**              |
| -------- | ---------- | ----------------------- | ----------------------- | ------------------------------- |
| 🔒        | **锁**     | **两边都有**            | **不许改** (防手滑)     | ✅ 会动                          |
| ✅        | **勾选框** | **左边** (Dope Sheet)   | **关掉动画** (Mute)     | ❌ **不动了**                    |
| 👁️        | **眼睛**   | **右边** (Graph Editor) | **看不见线** (Hide)     | ✅ **照样动**                    |
| 🔧        | **扳手**   | **左边** (Dope Sheet)   | **开关特效** (Modifier) | ✅ 会动 (但这取决于你开关的状态) |

**新手提示：** 最容易混淆的是 **“勾选框”** 和 **“眼睛”**。记住：**勾选框是“拔电源”（不动了），眼睛是“拉窗帘”（看不见）。**

#   大于0的部分小于0的部分，分别是什么意思？

![image-20260123193911577](C:\Users\Administrator\AppData\Roaming\Typora\typora-user-images\image-20260123193911577.png)

这个问题问到了 Graph Editor 的核心逻辑：**坐标系与数值**。

在这张图里，**横向**代表时间（第几帧），而**纵向**（上下）代表数值。

那条 **水平的蓝色粗线（0线）** 就是 **“原点”** 或者 **“初始状态”**。

### 1. 对于“位置” (Location) 通道

*(比如你的红线 X Location 和 绿线 Y Location)*

- **0 线 (中间):**
  - 代表 **世界中心原点** (0,0,0)。也就是你在 3D 视图里看到的红绿线交叉的那个中心点。
- **0 以上 (正数 +):**
  - 代表物体沿着坐标轴箭头的 **正方向** 移动。
  - *比如 X 轴：向“右”跑。*
- **0 以下 (负数 -):**
  - 代表物体沿着坐标轴箭头的 **反方向** 移动。
  - *比如 X 轴：向“左”跑。*

> **看你的图（红线）：**
>
> 你的红线 (X Location) 大幅弯到了 **0 以下**（大约 -15 左右）。
>
> **意思是：** 在这段时间里，你的罐子沿着 X 轴的**反方向**（也就是逆着红色箭头的方向）跑了 15 米远。

------

### 2. 对于“旋转” (Rotation) 通道

- **0 线:**
  - 代表 **没有旋转**，或者归零状态。
- **0 以上 (+):**
  - 代表 **逆时针** 旋转（通常）。
- **0 以下 (-):**
  - 代表 **顺时针** 旋转。

------

### 3. 对于“缩放” (Scale) 通道（特殊！）

- **注意：** 缩放的基准线通常不是 0，而是 **1**。
- **1:** 代表 **原始大小**。
- **0:** 代表 **缩没了**（变成一个看不见的点）。
- **0 以下 (-):** 代表 **翻转**（模型会里外翻面，很少用到）。

### 🚀 简单总结

你可以把中间的 **0线** 想象成 **“家”**。

- **线往上走：** 离家出走，往东边跑。
- **线往下走：** 离家出走，往西边跑。
- **线回到 0：** 回家了。

**想要验证一下吗？**

你选中罐子，看一眼 3D 视图右上角的彩色坐标轴（Gizmo）。

- **红色箭头**指的方向就是 X 轴的**正方向（0以上）**。
- 既然你的曲线是负的，你的罐子现在肯定是在红色箭头**相反**的那一头。

### 第一部分：主右键菜单 (F-Curve Context Menu)

这是最基础的菜单，涵盖了所有的操作入口。

1. **Copy (复制) `Ctrl C`**:
   - 将选中的关键帧数据复制到内存剪贴板。
2. **Paste (粘贴) `Ctrl V`**:
   - 将剪贴板中的关键帧粘贴到当前时间指针（Playhead）所在的位置。
3. **Paste Flipped (翻转粘贴) `Shift Ctrl V`**:
   - 粘贴关键帧，但**数值取反**。例如：原关键帧是 X轴旋转 30度，翻转粘贴后变成 -30度。常用于制作对称动作（如左脚迈步->右脚迈步）。
4. **Handle Type (控制柄类型) `V`**:
   - *(下文详细解释)* 用于调整关键帧手柄的形态，决定曲线是圆滑还是尖锐。
5. **Interpolation Mode (插值模式) `T`**:
   - *(下文详细解释)* 用于决定两个关键帧中间的“过渡算法”。
6. **Easing Type (缓动类型) `Ctrl E`**:
   - *(下文详细解释)* 用于决定加速/减速发生的时机（是开始时慢，还是结束时慢）。
7. **Insert Keyframes (插入关键帧) `I`**:
   - 在当前鼠标悬停的时间点，强制为当前选中的曲线添加一个关键帧。
8. **Duplicate (复制) `Shift D`**:
   - **原地**复制选中的关键帧，并自动进入“抓取移动”状态。比 Ctrl C/V 更快，适合在同一个通道内快速克隆动作。
9. **Delete Keyframes (删除关键帧) `X` 或 `Delete`**:
   - 删除选中的关键帧。
10. **Mirror (镜像) `Ctrl M`**:
    - 将选中的关键帧按照特定轴进行翻转。它通常包含两个隐藏逻辑：按时间翻转（倒放）或按数值翻转（上下颠倒）。
11. **Snap (吸附)**:
    - *(下文详细解释)* 用于将关键帧对齐到特定的数值或时间点。

------

### 第二部分：控制柄类型 (Handle Type) `V`

 ![image-20260123201827155](C:\Users\Administrator\AppData\Roaming\Typora\typora-user-images\image-20260123201827155.png)

这个菜单决定了**单个关键帧**两侧线条的弯曲方式。

#### 1.  **Free (自由)**:

   - **解释：** 左右两个手柄完全独立。
   - **效果：** 你可以把左边拉得很长（入点缓），右边拉得很短且角度不同（出点急）。用于制作极端的转折点，比如物体撞击墙壁后反弹的瞬间。

####  2.  **Aligned (对齐)**:

   - **解释：** 左右手柄在一条直线上，角度锁定，但长度可以不同。
   - **效果：** 保证曲线通过该点时是平滑的，没有折角。这是手动调节曲线弧度时的默认首选。

####  3.  **Vector (矢量/折线)**:

   - **解释：** 手柄自动指向前后的关键帧。
   - **效果：** 产生直线过渡。如果你移动了周围的关键帧，它会自动更新指向。常用于机械运动或不需要加减速的直接切换。

####  4.  **Automatic (自动)**:

   - **解释：** Blender 自动计算最平滑的曲线。
   - **效果：** 也就是默认的黄色手柄。
   - **缺点：** 容易产生“过冲”（Overshoot），即曲线在两个数值之间可能会稍微凸出去一点再回来。

####  5.  **Auto Clamped (自动钳制)**:

   - **解释：** 也是自动平滑，但增加了限制算法。

   - **效果：** **它保证曲线绝对不会超出前后两个关键帧的数值范围**。

   - **用途：** 这是制作摄像机静止、物体完全停稳时的**最佳选择**，防止物体在静止状态下发生微弱的漂移。

#### 这部分是对所有的手柄生效还是只对选中的手柄生效？

  答案非常明确：**只对你当前选中的（Selected）关键帧/手柄生效。**

这是 Blender 的一条底层逻辑：“**操作跟随选择**”。

##### 详细规则：

1. **如果你选中了一个关键帧（中间的小方块/菱形）：**
   - 该关键帧**左右两边**的手柄都会同时变成你选择的类型（比如同时变成“自由”或“自动”）。
   - **未被选中**的其他关键帧**完全不会**受影响。
2. **如果你想修改整条曲线：**
   - 你需要先按快捷键 **`A`** （Select All）全选所有关键帧。
   - 然后再按 `V` 选择类型，这样整条线才会变。
3. **为什么这样设计？（实战价值）**
   - **安全：** 你可能花了一下午把前半段动画调得很完美，现在只想修一下第 50 帧的一个小转折。因为“只对选中生效”，你可以放心大胆地修改第 50 帧，不用担心把前面调好的动画搞乱。

##### 总结

- **没选中 = 没反应。**
- **选中谁 = 改谁。**

### 第三部分：插值模式 (Interpolation Mode) `T`

![image-20260123201851538](C:\Users\Administrator\AppData\Roaming\Typora\typora-user-images\image-20260123201851538.png) 

这个菜单决定了**两帧之间**的计算逻辑。

#### **Interpolation (基础类)**

1. **Constant (常值)**:
   - **解释：** 保持前一帧的数值不变，直到遇到下一帧瞬间跳变。
   - **用途：** 类似“定格动画”或“瞬间移动”的效果。
2. **Linear (线性)**:
   - **解释：** 点对点连直线，匀速运动。
   - **用途：** 传送带、机械臂、循环背景滚动。
3. **Bezier (贝塞尔)**:
   - **解释：** 使用手柄控制的平滑曲线（默认模式）。
   - **用途：** 绝大多数自然且有物理感的动画。

#### **Easing (by strength) (缓动强度类)**

这些选项本质都是曲线，区别在于**加速/减速的剧烈程度**。从上到下，变化率越来越“陡峭”。

1. **Sinusoidal (正弦)**: 最柔和的起步和停止。
2. **Quadratic (二次方)**: 稍微快一点的加速。
3. **Cubic (三次方)**: 明显的加速感。
4. **Quartic (四次方)**: 强烈的加速感。
5. **Quintic (五次方)**: 极度剧烈的加速/减速（像火箭发射，起步极慢，后面极快）。
6. **Exponential (指数)**: 基于指数函数，比五次方更极端。
7. **Circular (圆形)**: 基于圆弧计算，启动时非常突然，或者结束时像撞墙一样突然停下。

#### **Dynamic Effects (动态特效类)**

1. **Back (回弹)**:
   - **解释：** 到达目标数值前，会稍微“冲过头”一点再缩回来；或者开始前先“向后蓄力”。
   - **用途：** 表现物体的冲力或预期动作（Anticipation）。
2. **Bounce (弹跳)**:
   - **解释：** 模拟皮球落地后的反弹波形。
   - **用途：** 物体掉落、文字砸向地面。
3. **Elastic (弹性)**:
   - **解释：** 像弹簧一样震荡。数值会大幅度越过终点，然后来回摆动几次。
   - **用途：** 软体动物、果冻效果、卡通风格的UI动效。

------

### 第四部分：缓动类型 (Easing Type) `Ctrl E`

![image-20260123201944557](C:\Users\Administrator\AppData\Roaming\Typora\typora-user-images\image-20260123201944557.png) 

这个菜单用于“修饰”上面的插值模式，决定特效发生在**哪一头**。

1. **Automatic Easing (自动缓动)**:
   - Blender 根据上下文自动判断，通常默认表现为 Ease Out 或 Ease In Out。
2. **Ease In (缓入)**:
   - **解释：** 效果发生在**动作开始**的时候。
   - **举例：** 如果选了 `Bounce` + `Ease In`，物体会在**起飞前**在地上弹几下再飞走（通常比较反直觉，用得少）。
3. **Ease Out (缓出)**:
   - **解释：** 效果发生在**动作结束**的时候。
   - **举例：** 如果选了 `Bounce` + `Ease Out`，物体会飞过来，落地时**弹跳停下**（这是最常用的用法）。
4. **Ease In and Out (缓入缓出)**:
   - **解释：** 动作的开始和结束都有效果。

------

###   第五部分：镜像（Mirror ）` Ctrl+M`

这个菜单的核心逻辑是：**你希望这些关键帧按照什么“中心点”进行翻转？**

在制作产品广告视频（你的目标方向）时，这个功能主要用于快速制作“倒放”效果或“反向”动作。

以下是每个子选项的详细拆解：

#### 1. By Times Over Current Frame (按当前帧镜像时间) `Ctrl M`

这是最常用的**“倒放”**工具。

- **含义：** 以你当前的时间指针（蓝色播放头）为中心轴，水平翻转关键帧。
- **逻辑：** 之前在左边的帧会被翻到右边，右边的翻到左边。
- **实战场景：**
  - 你做了一个“产品从包装盒里飞出来”的动画（0-50帧）。
  - 现在客户要求视频结尾要“产品飞回包装盒”。
  - 你只需要复制关键帧，把时间指针放在中间，选这个选项，动画就瞬间变成了倒放。

#### 2. By Values Over Cursor Value (按游标数值镜像数值)

这个比较冷门，它是以**2D游标（2D Cursor）**的高度为中心进行垂直翻转。

- **含义：** Graph Editor 里也有一个类似 3D 视图的游标（如果你按住 `Shift + 右键` 可以移动它）。这个选项会以那个游标的 Y 轴高度为镜子，上下翻转你的曲线。
- **实战场景：** 当你需要以一个非常特定的、非 0 的数值为中心做对称时使用。比如你的物体悬浮在高度 5米 的位置，你想让它以 5米 为中心上下波动，而不是以 0 为中心。

#### 3. By Times Over Zero Time (按 0 时间镜像时间)

- **含义：** 以第 0 帧为轴，水平翻转关键帧。
- **逻辑：** 第 10 帧的关键帧会跑到 -10 帧去。
- **实战场景：** 主要用于物理模拟或循环动画的初始设置。比如你需要给角色做一个“预备动作”，你可以把动作镜像到负数帧，这样动画一开始就是动作的中间状态。

#### 4. By Values Over Zero Value (按 0 数值镜像数值) **(核心常用)**

这是制作**“反向动作”**的神器。

- **含义：** 以 Y=0（中间的水平线）为轴，垂直翻转关键帧。正数变负数，负数变正数。
- **逻辑：** `+5` 变成 `-5`。
- **实战场景：**
  - **对称运动：** 比如你做了一个“摄像机向右摇（Rotation Z > 0）”的镜头，觉得不好看，想改成“向左摇”。不用重新 K 帧，直接选这个，摄像机立马反向。
  - **呼吸感：** 制作悬浮产品时，复制上升的曲线，用这个功能翻转，就能得到完美的下降曲线。

#### 5. By Times Over First Selected Marker (按首个选中标记镜像时间)

- **含义：** 以你在时间轴上打的**标记（Marker）**为中心轴进行翻转。
- **逻辑：** 你可以在时间轴上按 `M` 键打标记。选中一个标记和一堆关键帧，使用此功能，关键帧会围绕这个标记翻转。
- **实战场景：** **卡点剪辑**。如果你是根据音乐节奏做动画，你在“重音”处打了一个 Marker。你可以用这个功能让动作精准地在重音前后对称（比如重音前是聚拢，重音后是散开）。

#### 总结

对于你的 Freelancer 广告视频制作工作流：

- 死记 **By Times Over Current Frame**：用来做**倒放**（比如盖子打开 -> 盖子合上）。
- 死记 **By Values Over Zero Value**：用来做**反向**（比如向左转 -> 向右转）。
- 其他三个选项作为特殊情况下的储备知识即可。

# 这个区域，可以单方向的缩放吗？

可以。在 Blender 的曲线编辑器（Graph Editor）中，单方向缩放（只拉伸 X 轴或只拉伸 Y 轴）是极其常用的操作，否则你很难看清那些扁平的曲线细节。

有两种“缩放”的含义，我不确定你是指**“调整视图”**（让你看得更清楚）还是**“修改关键帧数据”**（改变动画幅度）。通常新手问的是第一种。

### 情况 1：仅调整视图（看得更清楚，不改变动画）

这是最常用的操作，因为默认情况下曲线经常缩成一条线，根本看不清。

- **快捷键大法（推荐）：**

  1. 鼠标悬停在曲线编辑器的区域内。
  2. **按住 `Ctrl` 键不放**。
  3. **按住鼠标中键（滚轮）** 并拖动：
     - **上下拖动：** 只拉伸 **Y 轴**（垂直视图）。能让你把原本平缓的曲线拉得“很陡”，方便看清微小的抖动。
     - **左右拖动：** 只拉伸 **X 轴**（时间视图）。

- **滚得条大法（备用）：**

  看你截图的右侧和底部都有滚动条。滚动条的两头各有一个**小圆点**。

  - 拖动那个**小圆点**，也可以单向拉伸视图。

------

### 情况 2：修改关键帧数据（真正改变动画）

如果你是想让动画变得“更快”或“动作幅度更大”：

1. 选中你要缩放的关键帧。
2. 按 **`S`** (Scale)。
3. 紧接着按 **`X`**：只在时间上缩放（调整动画快慢）。
4. 或者按 **`Y`**：只在数值上缩放（调整动作幅度，比如手抬得更高）。

#  Ctrl+H，是什么操作？

这是一个非常容易让新手“心脏骤停”的快捷键。

**`Ctrl + H`** 在 Blender 曲线编辑器中的功能是：**控制柄的显示与隐藏 (Show/Hide Handles)**。

它通常会弹出一个小菜单，或者直接切换状态，决定你是否能看到关键帧旁边的“小把手”（杠杆）。

## 为什么说它容易让人“心脏骤停”？

很多新手（可能也包括未来的你）在操作时手滑按到了 `Ctrl + H`，然后发现：

**“完了！我的曲线变成直线了？把手全没了！我怎么调节弧度啊？”**

其实曲线没变，只是“把手”被隐藏起来了。

## `Ctrl + H` 的菜单选项通常包含：

1. **All (全部显示)：** 默认状态。每个选中的关键帧都显示把手。
2. **None (全部隐藏)：** 只显示关键帧的点，不显示把手。
   - **用途：** 当你的关键帧密密麻麻挤在一起时，把手会互相遮挡乱成一锅粥。这时候隐藏把手，只看曲线的走势，视野会非常清爽。
3. **Selected Only (仅选中的)：** **(强烈推荐)**
   - **用途：** 只有当你鼠标选中某个关键帧时，它的把手才显示出来；没选中的关键帧就不显示。
   - **实战价值：** 这是最干净、最高效的工作模式，建议常驻。

------

## 区分一下“三H兄弟”：

为了防止混淆，请一定要分清这三个“H”的区别：

| **快捷键**      | **功能**                   | **作用对象**                              | **后悔药（恢复）**           |
| --------------- | -------------------------- | ----------------------------------------- | ---------------------------- |
| **`H`**         | **隐藏 (Hide)**            | 隐藏你选中的**整条曲线**                  | `Alt + H`                    |
| **`Shift + H`** | **独显 (Hide Unselected)** | 隐藏**除了选中以外**的所有曲线            | `Alt + H`                    |
| **`Ctrl + H`**  | **藏把手 (Hide Handles)**  | 隐藏关键帧上的**控制手柄** (曲线本身还在) | 再按一次 `Ctrl + H` 改回 All |

## **一句话总结：**

如果你发现**曲线还在，但“把手”没了**，别慌，按 **`Ctrl + H`** 选 **All** 就能救回来。

#  时间线的快捷键，与 曲线编辑器区域的快捷键是通用的吗？

这是一个非常省心的好消息：**是的，通用度高达 90%。**

Blender 的快捷键设计逻辑是非常统一的。一旦你练熟了曲线编辑器（Graph Editor），你会发现时间线（Timeline）和摄影表（Dope Sheet）简直就是“简略版”的曲线编辑器。

为了帮你区分，我把它们分为**“完全通用”**和**“根本不能用”**两类：

### ✅ 1. 完全通用的“肌肉记忆” (放心按)

这些快捷键在两个区域的作用**一模一样**，你可以无缝切换：

| **快捷键**          | **功能**             | **两个区域的表现**                                           |
| ------------------- | -------------------- | ------------------------------------------------------------ |
| **`G`**             | **移动 (Grab)**      | 都是抓取关键帧。按 `G` 后左右拖动，都是调整时间位置。        |
| **`S`**             | **缩放 (Scale)**     | 都是以当前播放头为中心进行缩放（让动作变快或变慢）。         |
| **`Shift + D`**     | **复制 (Duplicate)** | 都是复制关键帧。                                             |
| **`X`** / **`Del`** | **删除 (Delete)**    | 都是删除选中的关键帧。                                       |
| **`Home`**          | **适配视图**         | 都是让关键帧铺满屏幕。                                       |
| **小键盘 `.`**      | **聚焦选中**         | 都是最大化显示你当前选中的关键帧。                           |
| **`A`**             | **全选**             | 都是全选。                                                   |
| **`B`**             | **框选**             | 都是拉个框选择一堆帧。                                       |
| **鼠标中键**        | **平移视图**         | 操作手感完全一致。                                           |
| **`Ctrl` + 中键**   | **缩放视图**         | **注意：** 在时间线里，通常只能左右缩放（拉长时间轴），上下缩放效果不明显（因为没有数值高度）。 |
| **`T`**             | **插值模式**         | 在时间线选帧按 `T`，照样能改 Linear / Bezier，虽然你看不到曲线变化，但数据确实改了。 |

------

### ❌ 2. 最大的区别 (千万别混淆)

既然通用度这么高，那区别在哪？

**区别在于“Y轴”（数值/高度）。**

- **曲线编辑器 (Graph Editor)：** 是**二维**的。
  - **X轴 = 时间**
  - **Y轴 = 数值 (强度)**
  - 你可以按 `G + Y` 上下移动关键帧，改变动作幅度。
- **时间线 (Timeline / Dope Sheet)：** 是**一维**的。
  - **只有 X轴 = 时间**
  - **没有 Y轴！**
  - **陷阱：** 如果你在时间线里按 `G + Y`（上下移动），**什么都不会发生**（或者只是在不同的通道行之间跳跃，但这通常不是你想要的效果）。
  - **陷阱：** 按 `V` (手柄类型) 在时间线里通常**看不到视觉反馈**，因为时间线里没有“手柄”给你看。

### 💡 给你的实战建议

作为广告片制作者，建议你养成这样的工作流习惯：

1. **在时间线 (Timeline) 做“粗剪”：**
   - 用 `G` 拖来拖去，把动作发生的**时间点**（节奏）定好。比如第1秒开始动，第3秒停。
   - 这里只管“什么时候动”，不管“动得怎么样”。
2. **在曲线编辑器 (Graph Editor) 做“精修”：**
   - 当时间点定好了，再去曲线编辑器里，用 `Ctrl + E` (Ease Out) 或 `V` (Auto Clamped) 去修**动作的质感**。

**结论：**

把你刚学的 `G`, `S`, `X`, `Shift+D` 放心大胆地用到时间线上吧，完全通用！只有涉及“上下移动”或“手柄形状”的操作，才必须回曲线编辑器。

#   Shift+D复制关键帧，是什么意思，什么效果？

“复制关键帧”翻译成大白话就是：**把物体在某一瞬间的状态（位置、角度、大小），“克隆”到另一个时间点去。**

在做产品广告视频时，它的核心作用只有两个，但都非常重要：

## 1. 制造“完美循环” (Loop)

这是最常见的用法。

- **场景：** 你在做一个产品的 **360度展示动画**。
- **操作：**
  1. 第 0 帧：物体旋转是 0度。
  2. 你把第 0 帧**复制**。
  3. **粘贴**到第 100 帧（并改成 360度）。
- **效果：** 因为起点和终点的状态在逻辑上是“接得上”的，视频播放完会自动从头开始，观众根本看不出视频在哪里结束，是一个无限循环。

## 2. 制造“绝对静止” (Hold) —— **广告片必用**

这是通过复制关键帧“人造”出来的停顿。

- **场景：** 摄像机飞快地移动到产品面前，然后你希望它**死死地停住 2 秒钟**让观众看清 Logo，然后再飞走。
- **操作：**
  1. 摄像机在第 20 帧停下了。
  2. 选中第 20 帧，**复制**。
  3. 把时间线拉到第 60 帧（2秒后），**粘贴**。
- **效果：**
  - **第 20 帧** 是这个位置。
  - **第 60 帧** 还是完全一模一样的这个位置。
  - **结论：** 在 20-60 帧这中间，摄像机**纹丝不动**。这就是“Hold（定格）”。

------

## 实操中的两个方法：

**方法 A：传统的 `Ctrl + C` / `Ctrl + V`**

- **流程：** 选中帧 -> `Ctrl+C` -> **移动蓝色时间指针到别处** -> `Ctrl+V`。
- **特点：** 比较慢，但是**可以跨物体粘贴（比如把A的动作粘给B）**。

**方法 B：高效的 `Shift + D` (推荐)**

- **流程：** 选中帧 -> 按 `Shift + D` -> **鼠标直接左右拖动** -> 点击鼠标左键放下。
- **特点：** 极快！不用动时间指针，直接把关键帧像拖文件一样拖过去。**在同一个物体上做“定格”时，永远用这个。**



#   物体缩放，xyz三个轴都是同一个曲线，这个是特殊的，还是一般情况？

这是一个**一般情况**（General Case），尤其是在做这种产品展示动画时。

简单来说：**只要你希望物体保持原本的形状（不被压扁或拉长），XYZ 三个轴的曲线就必须完全一致。**

这里有两个层面的解释：

### 1. 为什么它们现在是一样的？（均匀缩放）

在 Blender 中，当你按下 **`S`** 键进行缩放时，默认是 **均匀缩放 (Uniform Scaling)**。

- 意思是：如果你把物体放大 2 倍，那么 X 轴变 2 倍，Y 轴变 2 倍，Z 轴也变 2 倍。
- 当你按下 **`I`** 打关键帧时，Blender 会把这三个相同的数值都记录下来。
- **结果**：在曲线编辑器里，因为三条线的数值和变化时间完全一样，它们就完美重叠在了一起，看起来像是一条线。

### 2. 什么时候会是“特殊”的？（非均匀缩放）

如果 XYZ 的曲线**不一致**，通常是为了追求特殊的动画效果，最常见的就是 **“挤压与拉伸” (Squash and Stretch)**。

想象一个皮球掉在地上的动画：

- **落地瞬间**：它会变扁。此时 **Z 轴缩小**（变矮），而 **X/Y 轴放大**（变胖）。
- **这种情况下**：你的曲线图里，Z 轴的线会往下走，XY 轴的线会往上走，它们就会**分道扬镳**。

### 总结

对于你的这个易拉罐广告：

- **保持一致（现状）**：这是**正确**的。因为易拉罐是刚体（硬的金属），它出现时应该保持工业产品的标准比例，不能一会儿变成椭圆，一会儿变扁。
- **如果不一致**：你的易拉罐看起来就会像果冻做的，或者像被压坏了。

**建议**：

既然你是做产品动画，保持这三条曲线完全重合是标准的做法。你可以点击左侧列表里的 `X Scale`、`Y Scale`、`Z Scale` 单独查看，确认它们是不是数值完全一样。

#   Pivot Point -轴心点

![image-20260124162138652](C:\Users\Administrator\AppData\Roaming\Typora\typora-user-images\image-20260124162138652.png)

[曲线编辑-轴心选项演示](https://shuaiqi-z.github.io/blender_study_records/Graph%20Editor%20Pivot%20Point%20show.html) 

这是 **曲线编辑器 (Graph Editor)** 里的 **“轴心点” (Pivot Point)** 设置。

它决定了当你对 **关键帧（曲线上的点）** 进行 **缩放 (S)** 或 **旋转 (R)** 操作时，是**以哪里为中心**进行的。

这个功能在调整动画节奏时非常关键，具体选项的含义如下：

## 1. Bounding Box Center (包围盒中心) —— 默认选项

- **含义**：以你选中的所有关键帧的“正中心”为基准。
- **场景**：如果你选中了一整段曲线按 **`S`** (缩放)，曲线会向中间收缩。这通常用于**整体减弱**动画幅度。

## 2. 2D Cursor (2D 游标) —— 最常用、最强大

- **含义**：以那个 **灰色的十字准星 (2D Cursor)** 为中心。
- **场景（这是个神技）**：
  - 假设你的动画是从第 0 帧开始的，你想让整个动画**变快一倍**。
  - 你可以把 2D 游标放在 (0, 0) 的位置。
  - 选中所有关键帧，选择这个模式。
  - 按 **`S`** -> **`X`** -> **`0.5`**。
  - **结果**：所有关键帧都会向第 0 帧靠拢，动画时间减半，且起始位置不变。如果你用默认的“包围盒中心”，动画缩放后会乱跑，不在第 0 帧开始了。

## 3. Individual Centers (各自中心)

- **含义**：每个选中的关键帧都以自己为中心缩放。
- **场景**：这主要用于**调整手柄**。如果你选中了多个关键帧，想同时把它们的贝塞尔手柄拉长（让过渡更平滑），用这个模式按 **`S`**，手柄会变长，但关键帧的位置不会动。

## 总结

你截图里红框选中的是 **Bounding Box Center**，这是默认状态。

如果你觉得调整曲线时，关键帧总是“乱跑”，大概率是因为你应该切换到 **2D Cursor** 模式，定住一个点再缩放。

#   Modifiers

![image-20260124182151584](C:\Users\Administrator\AppData\Roaming\Typora\typora-user-images\image-20260124182151584.png) 

这张截图展示的是 **F-Curve Modifiers（曲线修饰符）** 的添加菜单。

这是 Blender 动画中非常强大的功能，它能让你**不打关键帧**就自动生成复杂的动画效果（比如自动循环、自动抖动、自动旋转）。

下面我为你详细解释每一个选项和图标的功能：

### 1. 右上角的两个小图标

这两个是修饰符的**复制粘贴**工具，非常实用：

- **左边（带向上箭头/复制图表）**：**Copy Modifiers (复制修饰符)**。
  - 把当前这条曲线上的修饰符设置复制到剪贴板。
- **右边（带向下箭头/粘贴图标）**：**Paste Modifiers (粘贴修饰符)**。
  - 把刚才复制的修饰符，粘贴到另一条曲线上。
  - *场景*：你给 X 轴做了一个完美的“抖动效果”，想让 Y 轴也一模一样抖动？点左边的复制，选 Y 轴，点右边的粘贴。搞定。

------

### 2. Add Modifier (添加修饰符) 菜单参数

这里列出的是你可以给曲线“挂载”的自动程序：

#### **Generator (生成器)**

- **作用**：用数学公式生成曲线（通常是直线或多项式）。
- **场景**：做**螺旋桨、车轮**这种需要**无限旋转**的东西。你不需要每隔 100 帧打个点，只要加这个，设定斜率，它就会永远转下去。

#### **Built-In Function (内置函数)**

- **作用**：生成标准的数学波形（正弦波 Sine、余弦波 Cosine 等）。
- **场景**：做**呼吸灯**、**钟摆**、或者物体上下**悬浮漂浮**的循环运动。它生成的曲线非常圆滑、规律。

#### **Envelope (包络线)**

- **作用**：给原有的动画加一个“外壳”来控制幅度。
- **场景**：比如你做了一个无限循环的波浪，但你希望它**“从无到有”**慢慢开始动，或者**“慢慢停下来”**。用 Envelope 可以像音量推杆一样控制动画的强弱。

#### **Cycles (循环)** —— **【最常用】**

- **作用**：让现有的关键帧**自动重复**。
- **场景**：**走路循环 (Walk Cycle)**、**转盘展示**。
  - 你只需要做 0-30 帧的动画，加这个修饰符，它就会自动把 30 帧之后的动作填满，一直重复播放。

#### **Noise (噪波)** —— **【你的易拉罐可能需要这个】**

- **作用**：给曲线增加随机的杂乱波动。
- **场景**：**模拟手持摄像机**的抖动、飞机遇到的**气流颠簸**、或者物体悬浮时**不规则的微动**。
  - *建议*：给你的易拉罐悬浮动画加一点点 Noise，它看起来就不会像定在那里的死物，而是有“空气感”。

#### **Limits (限制)**

- **作用**：给数值设一个“地板”和“天花板”。
- **场景**：防止穿模。比如你用 Noise 让物体乱动，但怕它掉到地板下面去，就可以加个 Limit，强制 Y 轴不能小于 0。

#### **Stepped Interpolation (阶梯插值)**

- **作用**：把顺滑的曲线变成锯齿状的阶梯。
- **场景**：**定格动画 (Stop Motion)** 效果，或者**蜘蛛侠平行宇宙**那种抽帧风格。它会让物体看起来像是一顿一顿的。

### 总结建议

对于你现在的易拉罐产品动画：

1. **Noise (噪波)**：最值得尝试。加在旋转或位移上，给很小的数值，能做出逼真的“空气悬浮感”。
2. **Cycles (循环)**：如果你想让易拉罐一直自转展示，不需要打几百个关键帧，打一圈，然后加 Cycles 即可。

#  有没有什么操作，可以快速的清除某一帧上的所有的关键帧？某个类型的关键帧？

这是一个非常实用的提问。在实际工作中，我们经常需要“反悔”，快速撤销某一帧的动作。

有非常快的方法，不需要一个个点右键。

### 1. 快速清除**某一帧上的所有关键帧** (Location + Rotation + Scale)

如果你想让物体在**这一帧**彻底变回“自由身”（没有任何关键帧锁定），最快的方法是利用 **Timeline (时间线)** 或 **Viewport (视口)** 快捷键。

- **方法 A：视口快捷键 (最快)**

  1. 在 3D 视图窗口中。
  2. 按下 **`Alt + I`**。
  3. Blender 会弹出一个菜单，选择 **Delete Keyframe (删除关键帧)**。

  - *逻辑：* 这会删除当前选中的物体在当前帧的所有活跃关键帧（如果你之前是用 I 键打的整套，它就会删整套）。

- **方法 B：时间线“一锅端” (最直观)**

  1. 看底部的 **Timeline (时间线)** 面板。
  2. 你会看到当前帧的位置有一排垂直的 **◇ (菱形)**。
  3. 最上面那一行（Summary / 汇总行）的菱形是白色的。
  4. **选中这个白菱形**（它代表了这一帧下面所有的 X,Y,Z 关键帧）。
  5. 按 **`X`** 或 **`Delete`** 键。

  - *结果：* 这一帧的所有动作瞬间清空。

------

### 2. 快速清除**某个类型**的关键帧 (比如只删位移，保留旋转)

如果你只想删除 **Location (位置)** 的关键帧，但是想保留 **Rotation (旋转)** 的动作，**N 面板** 是最高效的战场。

- **技巧：鼠标悬停在“大标题”上**

  不要去对着 X、Y、Z 一个个点。请看 N 面板上的文字标签。

  1. **鼠标悬停** 在 **"Location" (位置)** 这个**文字单词**上 (不要悬停在具体的 X/Y/Z 数值上)。
  2. **右键点击** -> 选择 **Delete Keyframes** (或者直接按快捷键 **`Alt + I`**)。

  - **效果：** Blender 会一次性把 X、Y、Z 三个轴的位置关键帧全部删掉。
  - **同时：** 下面的 "Rotation" (旋转) 因为你没指它，所以完好无损。

### 3. 总结表格 (建议背诵 `Alt + I`)

| **你的需求**         | **鼠标位置**                  | **操作 / 快捷键**             | **效果**                 |
| -------------------- | ----------------------------- | ----------------------------- | ------------------------ |
| **删当前帧的一切**   | 3D 视图空白处                 | 按 **`Alt + I`**              | 删除该物体当前帧所有动画 |
| **只删“位置” (XYZ)** | N面板 **"Location"** 文字上   | 按 **`Alt + I`** (或右键删)   | 位置清空，旋转/缩放保留  |
| **只删“X轴位置”**    | N面板 **"X Location"** 数值上 | 按 **`Alt + I`** (或右键删)   | 只删 X 轴，Y/Z 保留      |
| **删这一帧的一切**   | 时间线 (Timeline)             | 选中**最顶端菱形** -> **`X`** | 视觉上最放心的方法       |

**一句话口诀：**

想要删谁，鼠标就指着谁（指物体删全部，指标题删一组，指数值删一个），然后按 **`Alt + I`**。

#   blender  5.0 timeline区域，x键出现的面板，解释所有选项。

![image-20260125104803006](C:\Users\Administrator\AppData\Roaming\Typora\typora-user-images\image-20260125104803006.png) 

这张截图展示的是 Blender **时间线 (Timeline)** 或 **摄影表 (Dope Sheet)** 中按下 **`X`** 键后弹出的删除菜单。

这是你管理动画数据卫生的“扫帚”。虽然有三个选项，但它们“打扫”的力度完全不同。

以下是每个选项的详细解释：

### 1. Delete Keyframes (删除关键帧) —— **最常用**

- **含义：** 仅仅把你**当前选中**的那几个菱形（关键帧）删掉。
- **形象比喻：** 像是从书架上拿走了一本书。书架（通道名字）还在，只是空了。
- **实战场景：**
  - 你按错键了，这一帧不需要动作，选中它删掉。
  - **这是你 99% 的时间都在用的功能。**

### 2. Clean Keyframes (清理关键帧) —— **优化数据**

- **含义：** 它会自动帮你判断并删除**“没用的废话”**。
- **逻辑：** 如果第 1 帧是 `10`，第 5 帧是 `10`，第 10 帧还是 `10`。中间那个第 5 帧就是“废话”，因为它没有任何变化。这个功能会把中间那个删掉，保留头尾。
- **实战场景：**
  - 当你从其他软件导入动画，或者“烘焙 (Bake)”了物理模拟动画时，每一帧都密密麻麻全是点。用这个功能可以瞬间把数据量减小，只保留必要的转折点，方便你后续修改。

### 3. Clean Channels (清理通道) —— **连根拔起**

- **含义：** 删除**所有选中的动画通道 (F-Curves)**，哪怕里面还有关键帧。或者用于清理那些已经没有关键帧的空通道。
- **形象比喻：** 这一次不是拿走书，而是把**整个书架都拆了扔出去**。
- **关键区别：**
  - 如果你只用 `Delete Keyframes` 删光了某行的关键帧，那个属性的名字（比如 "X Location"）还会留在左侧列表里，只是没数据了。
  - 如果你用 `Clean Channels`，这个属性的名字会**直接从左侧列表中消失**。
- **实战场景：**
  - **强迫症福音：** 你不小心手滑给“缩放 (Scale)”打了个关键帧，然后删了关键帧，但“Scale”这行字死赖在左边不走，看着心烦。这时候选 "Clean Channels"，它就彻底消失了。

------

### 💡 广告人避坑指南

- **平时干活：** 闭眼选 **Delete Keyframes**。
- **收尾整理：** 如果左边列表太乱，有很多空的或者不想要的属性条目，用 **Clean Channels** 清理一下。
- **千万别乱点：** 别在没选对东西的时候点 `Clean Channels`，它有时候脾气比较暴躁，会把你不想删的轨道连根拔起（如果判定为无效的话）。**撤销 (`Ctrl + Z`) 是你的好朋友。**

#   同一个位置打关键帧以后，在做更改，在打关键帧是重新打还是修改？

这是一个非常关键的概念，直接关系到你的工作效率和会不会“白干”。

简单直接的回答：**是修改（覆盖）**。

在 Blender 中，同一个通道在同一帧上，只能有一个数值。你不需要“先删旧的再打新的”，直接打就是**覆盖**。

------

### 1. 它是怎么运作的？（覆盖逻辑）

想象每一个关键帧就是一个**“座位”**。

- **原来：** 第 10 帧的座位上坐着“5米”。
- **操作：** 你把物体移到“8米”，再次在第 10 帧按 `I`。
- **结果：** Blender 会把“5米”直接踢走，让“8米”坐在这个座位上。
- **结论：** 不需要删除，直接按 `I` 就是更新数据。

### 2. 千万要看懂的“颜色红绿灯” 🚦

在 N 面板（侧边栏）里，颜色的变化会告诉你现在的状态，这比你记忆操作更靠谱：

- **🟨 黄色 (Yellow)：**
  - **含义：** **数据已保存**。
  - *状态：* 你刚按完 `I`，或者这里本来就有关键帧且你没动它。这时候是安全的。
- **🟧 橙色 (Orange)：** **(危险区域！)**
  - **含义：** **你改了，但还没存！**
  - *场景：* 第 10 帧本来是 5米（黄色）。你把它拖到了 8米，这时候数值背景会瞬间变成**橙色**。
  - *警告：* 如果你这时候**不按 `I`**，直接拖动时间线去别的地方，物体会**瞬间弹回** 5米。你的修改就丢了。
  - *正确操作：* 看到橙色，如果不想要就切走；如果想要，立刻按 `I` (或右键插帧) 把它变回黄色。

### 3. 给 Freelancer 的效率建议：自动插帧 (Auto Keying)

每次改完都要按 `I` 确实很累，还要防止忘按导致白干。

- **推荐操作：** 打开时间线底部的 **🔴 圆点按钮 (Auto Keying)**。
- **效果：**
  - 只要你的时间线停在有关键帧的地方。
  - 你随便拖动一下物体（变橙色）。
  - 松开鼠标的瞬间，Blender 会**自动帮你按 `I`**（变黄色）。
  - **这就是“自动覆盖”模式**，做动画调整时效率极高。

**总结：**

不用删，**直接打就是改**。但一定要盯着数值颜色：**变橙色如果不按 `I`，一动时间线就“白改了”。**

#   跟随路径，为啥不是与路径100%重合？

![image-20260125152945823](C:\Users\Administrator\AppData\Roaming\Typora\typora-user-images\image-20260125152945823.png) 

这是一个非常经典的 Blender“灵异事件”，困扰过无数新手。

简单直接的原因是：**你的物体身上自带了“额外”的位移数据。**

Blender 的逻辑是这样的：

**最终位置 = 路径给的位置 + 你自己原本的位置 (Offset)**

现在的状况是：路径（Path）确实在拉着你的物体走，但是因为你的物体本身在 X/Y/Z 轴上不是 `(0,0,0)`，所以它就像是**“被一根隐形的棍子撑开了一段距离”**陪着路径跑，而不是踩在路径上跑。

### ✅ 1秒钟修复法

请按照这个步骤操作，立马就能吸附回去：

1. **选中** 那个正在跟随路径的物体（比如你的摄像机或罐子）。
2. 按下快捷键 **`Alt + G`** (重置位移)。
   - *解释：* 这会把你物体身上的“额外位移”清零。
3. **见证奇迹：** 物体会瞬间“啪”地一下，吸附到那条虚线路径上，严丝合缝。

------

### 🕵️‍♂️ 为什么会这样？(避坑原理)

当你给物体添加“跟随路径 (Follow Path)”约束或者设置父级时，Blender 默认是**保留你当前的相对距离**的。

- **正确做法：** 应该先把物体放在世界中心 `(0,0,0)`，或者把物体放在路径的起跑点上，再去加约束。
- **错误做法（现状）：** 你可能是在物体离路径很远的时候加的约束，或者加完约束后手滑按 `G` 移动了它。这时候，这个距离就被当作“故意保留的偏移量”记录下来了。

**总结：**

以后只要发现物体**“虽然跟着跑，但是离得八丈远”**，直接 **`Alt + G`** 准没错！

#   graph  editor中，有什么可以屏蔽某个通道某个关键帧的操作吗？

在 Blender 的 Graph Editor（曲线编辑器）中，针对“屏蔽”（即让它不起作用）的需求，分两种层级：**屏蔽整个通道** 和 **屏蔽单个关键帧**。

Blender 的逻辑是：**通道可以屏蔽，但单个关键帧不能“屏蔽”，只能删除或修改。**

以下是具体的操作方法：

### 1. 屏蔽整个通道 (Mute Channel)

如果你觉得“X轴旋转”看起来怪怪的，想暂时关掉它看看效果：

- **操作方法：** 在左侧通道列表中，**取消勾选** 通道名字前面的**复选框**。
- **快捷键：** 选中通道名字（比如 X Rotation），按下 **`Shift + W`** (Toggle Channel Mute)。
- **视觉变化：** 对应的曲线会变成**灰色**，这代表它被“静音”了，不再对物体产生任何影响。再次按 `Shift + W` 可恢复。

### 2. 屏蔽单个关键帧 (Mute Keyframe)

**遗憾的是，Blender 没有直接“静音单个关键帧”的功能。**

因为曲线是连续的，如果你让中间一个点“失效”，电脑不知道你是想让曲线“跳过”这个点直接连直线，还是保持上一个状态。

**替代方案（如何测试某个关键帧是不是罪魁祸首）：**

- **方案 A：删除大法 (最快)**

  1. 选中那个关键帧。
  2. 按 **`Del`** 或 **`X`** 删除它。
  3. 播放动画看效果。
  4. 如果是误杀，立刻按 **`Ctrl + Z`** 撤销回来。

- **方案 B：打平手柄 (Flatten Handles)**

  如果你不想删除，只是不想让它产生剧烈的起伏：

  1. 选中那个关键帧。
  2. 按 **`S`** (缩放)，然后按 **`Y`** (锁定Y轴)，输入 **`0`**，回车。
  3. 这会把它的手柄拉平，让动作变得极其平缓，接近于“没有变化”。

### 3. 另一种“屏蔽”：锁定 (Lock)

如果你说的“屏蔽”是指**“保护它不被我不小心改动”**（但动画照常播放）：

- **操作方法：** 点击通道名字旁边的 **小锁头图标** (Padlock)。
- **快捷键：** 选中通道，按 **`Tab`** 键。
- **效果：** 曲线还在动，但你无法移动上面的任何关键帧，怎么拖都拖不走。这是防止手滑的最佳操作。

### 总结建议

- 怀疑某个**轴**有问题？ -> **`Shift + W` (屏蔽通道)**。
- 怀疑某个**点**有问题？ -> **`Del` 删除看看，不对就 `Ctrl + Z`**。
- 不想手滑改坏了？ -> **`Tab` (锁定通道)**。

#  M键的这些标记是什么作用？

![image-20260126143228016](C:\Users\Administrator\AppData\Roaming\Typora\typora-user-images\image-20260126143228016.png) 

这是 Blender 中的 **时间线标记 (Timeline Markers)**。

就是一条线不会有任何影响。

你可以把它们理解为你在看书时贴的**“书签”**或者**“便利贴”**。

### 它们的作用是：

1. **记笔记 (注释)：**

   用来提醒你自己“这一帧发生了什么”。

   - 图中 `F_21` 和 `F_22` 很可能是指“第21帧”和“第22帧”（Frame 21/22）。
   - **注意看曲线：** 在这两个标记之间（21帧到22帧），你的曲线有一个**垂直的跳变**。这说明这里发生了一个瞬间的突变（比如剪辑点、瞬间位移）。这两个标记就是专门标出来告诉你：“注意！这里有个关键节点”。

2. **相机切换 (高阶用法)：**

   在做多镜头动画时，我们可以把相机绑定到标记上。比如到了 `F_21` 这个标记，视角自动切换到“顶视图相机”。

### 常用操作（你可以试试）：

- **添加标记：** 鼠标放在时间轴上，按 **`M`** 键。
- **重命名：** 选中标记（标记会变白），按 **`Ctrl + M`**。你可以把它改成“开始旋转”或“落地”。
- **移动标记：** 选中后按 **`G`** 拖动。
- **删除标记：** 选中后按 **`X`** 或者 `Delete`。

**对你的影响：**

它们**不会影响**动画的渲染结果，纯粹是给你（操作者）看的辅助线。如果你觉得它们碍眼，直接删掉 (`X`) 完全没问题。
