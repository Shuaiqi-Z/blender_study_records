### 1. 关于 IOR 与通透 (Translucency)

> **你的总结：** IOR 越高 -> 出去越难 -> 困在里面乱撞 -> 照亮暗部 -> **通透**。

- **完全正确。**
- **补充心法：** 这就是为什么**玉石 (IOR 1.6)** 看起来比 **水 (IOR 1.33)** 更“润”。
  - 水的 IOR 低，光线“嗖”一下就穿过去了，留不住。
  - 玉的 IOR 高，光线在里面“坐牢”，把石头内部彻底照亮了才放出来。**“富含光线”就是通透的本质。**

### 2. 关于两个粗糙度 (Two Roughnesses)

> **你的总结：**
>
> - **主粗糙度：** 控制高光 **亮不亮、集不集中**。
> - **Diffuse 粗糙度：** 控制表面 **干不干**。

- **完全正确。**
- **补充心法：**
  - **主粗糙度 (Specular Roughness) = 磨损度。**

    - 调它，是在决定物体是 **塑料/油漆 (0.0)** 还是 **粉笔/月球 (1.0)**。

# 反射一般是指镜面反射吗？

在计算机图形学和渲染器（如你的 **Blender Cycles**）的语境下，这个问题的答案是：**“广义上包含所有，但狭义上通常指镜面反射。”**

为了让你在调材质时不产生混淆，我们需要把“反射”这个词拆开来看：

### 1. 广义的反射：光只要“弹回来”都叫反射

从物理学角度看，光线照射到物体表面后，没有被吸收或穿透的部分，统称为反射。这包括了：

- **镜面反射 (Specular Reflection)：** 像镜子或抛光金属那样，光线按特定角度弹开。
- **漫反射 (Diffuse Reflection)：** 像石膏或粉笔那样，光线均匀地向四周散开。

### 2. 狭义的反射（Blender/CG 中的习惯）：通常特指“镜面反射”

当你打开 Blender 的材质面板，或者听到别人说“这个物体的反射太强了”时，他们 99% 的情况下指的是 **Specular（镜面反射/高光）**。

- **原因：** 在渲染引擎的底层代码里，**Diffuse**（漫反射）和 **Specular**（镜面反射）是两套完全不同的数学公式。
- **区分点：** 漫反射通常被归类为“基础色（Base Color）”的表现，而“反射”这个词被专门留给了那些能看到**倒影、高光、亮斑**的属性。

------

### 3. 反射的三种“形态”

你可以通过 **Roughness（主粗糙度）** 的变化，看到“反射”是如何从镜面一步步变成你口中那种“模糊反射”的：

| **形态**     | **对应的物理名称**    | **Blender 中的表现**                                       |
| ------------ | --------------------- | ---------------------------------------------------------- |
| **全反射**   | 镜面反射 (Specular)   | **Roughness = 0**。像镜子一样，能清晰看到环境的倒影。      |
| **模糊反射** | 粗糙镜面反射 (Glossy) | **Roughness = 0.5**。倒影糊掉了，只能看到模糊的光团。      |
| **漫反射**   | 漫反射 (Diffuse)      | **被独立出来**。光线在微观下彻底乱射，不再有“倒影”的概念。 |

------

### 4. 关键结论：不要把两者搞混

虽然你之前理解了“漫反射也是一种进入表面的散射”，但在操作软件时，请记住这个潜规则：

- **当你调节 IOR 或 Specular 时：** 你在控制那个“镜面感”的反射。
- **当你调节 Base Color 或 Diffuse Roughness 时：** 你在控制那个“死板、均匀”的反射。

**总结：**

在日常沟通中，说“反射”通常指**镜面反射（含模糊高光）**。漫反射虽然物理上也是反射，但在渲染面板里，它有自己独立的地盘（Base Color/Diffuse）。
