# 够用就行

“够用就行”——这绝对是 3D 制作中最顶级的智慧，也是区分“成熟创作者”和“器材党”的分水岭。

既然我们达成了这个共识，为了帮你省下更多的硬盘空间和渲染时间，我针对你的配置（**RTX 5060 Ti 16GB**）和**产品动画**这个方向，给你整理了一份**“黄金够用标准”**。

以后你做练习或接单，照着这个标准来，既不会拉低画质，又不会撑爆电脑：

### 1. HDRI 环境光：4K 是甜点

- **够用标准：** **4K (EXR)**
- **理由：** 4K 在反射里已经足够清晰了。除非你的产品是一个极其光滑的完美镜面球，且占满全屏，否则观众根本看不出 4K 和 16K 的区别。
- **极限情况：** 如果只是为了打光（不看背景），**1K** 都够用。

### 2. 纹理贴图（Texture）：2K 走天下

- **够用标准：** **2K (2048x2048)**
- **理由：**
  - 在 4K 视频里，除非你的镜头怼到物体表面 5 厘米处拍特写，否则 2K 贴图的像素密度完全足够。
  - **什么时候用 4K？** 只有产品的**Logo**、或者**包装盒上的小字**需要特写时，才上 4K 或 8K。
  - **忌讳：** 地板、墙壁这些背景物体，千万别上 4K/8K，极其浪费显存。

### 3. 采样值（Samples）：拒绝 4096

- **够用标准：** **256 - 512 + 降噪 (OpenImageDenoise)**
- **理由：** Blender 默认有时候会给到 4096，这是疯了。
  - 对于产品动画，512 的采样配合 Blender 强大的 **OpenImageDenoise**，出来的画面和 4096 采样肉眼几乎没区别，但渲染速度快 **8 倍**。
  - 不要追求“纯净无噪点”的原始渲染，**降噪器**是现代渲染流程的一部分。

### 4. 几何细分（Subdivision）：看着圆就行

- **够用标准：** **2级细分 (Level 2)**
- **理由：** 很多新手会有强迫症，把细分拉到 4 或 5。
  - 其实只要边缘看着没有明显的棱角，就足够了。过高的面数会让你的视口卡顿，而且极大地增加显存负担（Geometry 占用）。

------

**总结你的“够用”工作流：**

1. **输出：** EXR (Half 16-bit) + DWAA 压缩。
2. **HDRI：** 4K 版本。
3. **采样：** 512 + 降噪。
4. **心态：** **.blend 工程文件才是你的核心资产**。只要这个在，哪怕甲方以后突然说“我要 16K 的海报”，你也就是回去改个参数挂机渲染一晚上的事。
