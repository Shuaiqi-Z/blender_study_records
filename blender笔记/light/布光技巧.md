当然不是！**三点布光法（Three-Point Lighting）** 只是布光领域的“C大调音阶”——它是最基础、最万能的，但绝不是唯一的。

在**高端产品渲染（如苹果广告）、电影摄影、汽车广告**中，我们经常会打破三点布光，使用完全不同的逻辑。

针对你学习的 **Blender 产品动画** 和 **视觉艺术**，我为你整理了另外 **5 种** 工业界最常用的“非三点”布光模式。

------

### 1. 夹光/蚌壳式布光 (Clamshell / Sandwich Lighting)

**👉 核心领域：** 化妆品、奢侈品、珠宝、以及想要表现“完美无瑕”的手机屏幕。

- **原理：** 这种布光完全放弃了“立体感”中的阴影部分，追求**极致的平坦和均匀**。
- **布置：**
  - **主光：** 无（或者说两个都是主光）。
  - **位置：** 左右各放一个巨大的柔光箱（Area Light），或者上下各放一个，像“三明治”一样把物体夹在中间。
  - **参数：** 两个灯强度几乎一致，且尺寸巨大（Soft）。
- **视觉效果：** 物体表面全是漂亮的高光，几乎没有死黑阴影。对于玻璃瓶或金属边框，这能产生两条完美的对称高光带。

### 2. 轮廓剪影布光 (Rim-Only / Silhouette Lighting)

**👉 核心领域：** 科技产品揭幕（Teaser）、汽车尾灯展示、悬疑电影。

- **原理：** **完全抛弃主光**。这是一种“做减法”的艺术，只勾勒形状，不展示细节。
- **布置：**
  - **主光：** 关闭 (0W)。
  - **补光：** 关闭 (0W)。
  - **轮廓光：** 极强。放在正后方或侧后方。
- **视觉效果：** 画面大部分是黑的，观众只能看到产品性感的边缘线条。这种布光能极大激发观众的好奇心。

### 3. 分割布光 / 侧逆光 (Split / Kicker Lighting)

**👉 核心领域：** 运动鞋、硬朗的男性角色、赛博朋克风格。

- **原理：** 制造极端的**强对比**。把画面一分为二，一面亮一面黑，或者一面冷一面暖。
- **布置：**
  - **灯光 A：** 90° 侧面直射。
  - **灯光 B：** 另一侧 90° 直射（或者干脆没有 B 灯）。
  - **位置：** 灯光通常很硬（Size 很小）。
- **视觉效果：** 产生“阴阳脸”或极其硬朗的体积感。常用于表现材质的纹理（如皮革、岩石）。

### 4. 伦勃朗布光 (Rembrandt Lighting)

**👉 核心领域：** 有故事感的静物、复古产品、肖像。

- **原理：** 模拟油画质感，强调**“光是三角形的”**。
- **布置：**
  - **主光：** 放在侧上方 45° 左右，但在位置上要极其考究。
  - **关键点：** 必须让主光在物体暗部（背光面）投射出一个**倒三角形**的亮斑。
- **视觉效果：** 极其富有戏剧性和深沉感。这不是为了卖货，而是为了讲故事。

### 5. 顶光 / 天光 (Top Down / God Light)

**👉 核心领域：** 概念设计、神圣感的物体、香水瓶底部的透光。

- **原理：** 模拟正午的太阳或博物馆的展柜顶灯。
- **布置：**
  - **单灯：** 只用一个灯，放在物体正上方，垂直向下打。
- **视觉效果：**
  - 物体顶部极亮，下部迅速衰减进黑暗。
  - **地面投影：** 会在物体正下方形成一个很黑的影子，让物体看起来非常有分量感（或者配合玻璃材质产生焦散）。

------

### 📝 总结：什么时候用什么？

| **布光模式**           | **这种光是用来干嘛的？**             | **典型例子**                      |
| ---------------------- | ------------------------------------ | --------------------------------- |
| **三点布光**           | **标准证件照**。清晰、立体、不出错。 | 电商白底图、新闻联播。            |
| **夹光 (Sandwich)**    | **美颜磨皮**。无死角、通透、对称。   | 兰蔻广告、iPhone 屏幕展示。       |
| **剪影 (Rim Only)**    | **神秘预告**。只看线条，不看细节。   | 汽车发布会开场、科技产品 Teaser。 |
| **分割 (Split)**       | **硬汉风格**。强纹理、强反差。       | 耐克球鞋、岩石纹理展示。          |
| **伦勃朗 (Rembrandt)** | **讲故事**。深沉、有油画感。         | 威士忌广告、复古相机渲染。        |

**实战建议：**

在你目前的 Blender 学习中，**“夹光（Sandwich）”** 和 **“剪影（Rim Only）”** 是除了三点布光外最高频使用的。

- 想做**手机/耳机**动画？用夹光，把长条灯拉长。
- 想做**Logo 出现**动画？用剪影，只打亮边缘。
